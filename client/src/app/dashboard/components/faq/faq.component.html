FAQ

Wie konfiguriere ich ein MUD?

Nach dem Login drücken Sie rechts oben auf "MUD Erstellen". Zuerst wählen Sie einen Namen, den man links oben eintragen kann.
Danach klicken Sie sich durch die einzelnen Kategorien (Rassen, Klassen, etc.) und füllen die Felder aus, woraufhin Sie auf hinzufügen klicken.
Zum Schluss drücken Sie rechts oben auf "Spiel speichern" und schon ist ihre Konfiguration bereit zum Spielen.

Wie erstelle ich eigene Befehle?

Sobald Sie am Konfigurieren des Spiels sind, wechseln Sie auf das "Befehle" Fenster. Dort kann man bei "Typ" den Typ "Eigener Befehl" auswählen.
Dann wählt man zuerst den Namen des Befehls, welchen man bei "Befehl" einträgt.
Daraufhin kann man aus den Standardbefehlen einen eigenen Befehl erstellen (z.B Command 1: "move player north", Command 2: "look"). Diese werden nacheinander ausgeführt.

Welcher Befehl macht was?

"move player north": Spieler geht in den nördlich-angrenzenden Raum
"move player east": Spieler geht in den östlich-angrenzenden Raum
"move player south": Spieler geht in den südlich-angrenzenden Raum
"move player west": Spieler geht in den westlich-angrenzenden Raum
"look": Spieler schaut sich im Raum um
"inventory": Zeigt an, welche Gegenstände der Spieler bei sich trägt
"pickup [Itemname]": Lässt den Spieler einen Gegenstand aufheben
"hit [Spielername]": Spieler greift an
"inspect [Objektname]": Zeigt genauere Informationen über ein Item oder einen NPC
"drop [Itemname]": Entfernt ein Item aus dem Inventar
"use [Itemname]": Der Charakter benutzt das konsumierbare item
"stats": Zeigt die aktuellen Stats des Charakters an
"clear": Löscht alle Ausgaben auf der Konsole

Masterbefehle:
"invite list": zeigt alle Spieler an, die eine Beitrittsanfrage gestellt haben
"invite accept [Spielername]": Akzeptiert die Beitrittsanfrage des Spielers
"invite reject [Spielername]": Lehnt die Beitrittanfrage des Spielers ab
"invite add [Spielername]": Sendet eine Spieleinladung an einen Spieler
"kick [Spielername]": Wirft den Spieler aus dem Spiel
"hit [Spielername] [Raum] [Schaden]": Fügt einem Spieler Schaden zu
"heal [Spielername] [Raum] [HP]": Fügt dem Spieler oder NPC Lebenspunkte (HP) hinzu
"give [Spielername] [Raum] [Item]": Gibt dem Spieler oder NPC das Item
"spawn [Typ] [Raum] [Objektname]": Fügt einem Raum ein Item oder NPC hinzu
"teleport [Spielername] [Raum]": Teleportiert einen Spieler in den angegebenen Zielraum
"createroom [Options] [Raumname] [Nachricht]": Fügt einen neuen Raum hinzu. Am Platzhalter [Options] kann mit -n, -o, -s oder -w der Nachbarraum (Namen) angegeben werden
"deleteroom [Raumname]": Löscht einen Raum, sofern dieser leer ist

Wie trete ich einem Spiel bei?

Nach der Anmeldung werden die verfügbaren Spiele angezeigt. Möchte man einem dieser Spiele beitreten, so klickt man auf "Beitrittsanfrage senden".
Jetzt muss man auf die Zulassung des Masters warten. Sobald dieser den Beitritt erlaubt, kann man dem Spiel beitreten

Wie spielt man das Spiel?

Ist man dem Spiel beigetreten, so kann man mithilfe der Befehle spielen. Gibt man einen ungültigen Befehl ein, so wird eine Übersicht mit allen Befehlen angezeigt.
Diese kann man dann in beliebiger Reihenfolge ausführen und sich so durch das Spiel durchkämpfen.

Was kann ein Master tun?

Als Master hat man die Kontrolle über das gesamte Spiel. Man kann z.B. Spieler einladen, Spieler entfernen, Items oder NPCs erscheinen lassen und vieles mehr.
